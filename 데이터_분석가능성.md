## 🔍 **데이터 분석 전략**

### Phase 1: 문제의식 입증 (1-2장)
```
데이터: 영재특목자사진학율.xlsx + 위치 데이터
분석:
- 지역별 영재고/과학고 진학자 비율
- 시각화: 구별 원형 그래프 또는 막대 그래프
- 결론: "서구·유성구 편중" 정량화
```

### Phase 2: 자원 격차 입증 (3-7장)
```
데이터: 동아리 수, 교원 현황, 학교 위치
분석:
- 구별 평균 동아리 수
- 구별 과학·수학 교원 수
- 위도·경도로 학교 밀도 지도화
- 결론: 서구·유성구에 자원 집중
```

### Phase 3: AI 솔루션 설계 (10-13장)
```
1단계 (인프라 점수화):
- 각 구별 "교육 자원 점수" = 
  * (과학고 수 × 100 + 영재고 수 × 100 + 특성화고 수 × 50 
     + 평균 동아리 수 × 10 + 과학 교원 수) / 학생 수

2단계 (맞춤 추천):
- 학생 위치 + 관심 분야 입력
- 근처 학교의 관련 프로그램 추천
- 교육 부족 지역은 우선순위 상향

3단계 (이동 지원):
- 에듀버스: 동구·중구 학교 픽업 → 서구·유성구 프로그램

데이터: 모든 위치 데이터 활용
```

## 📌 **핵심 분석 지표 설계**

### 구별 "교육 기회 지수" 계산
```
지역 A의 교육기회지수 = 
  (영재고 수 / 지역 중학생 수) × 100
  + (과학고 수 / 지역 중학생 수) × 100
  + (평균 동아리 수) × 1
  + (과학 교원 수 / 전체 교원 수) × 50
  + (지하철 역 수) × 5
```

### 예상 결과
```
서구: 높은 지수 (연구소 집중, 학원 많음, 교원 풍부)
유성구: 높은 지수 (대덕연구단지, 과학고 집중)
동구: 낮은 지수 (자원 부족)
중구: 낮은 지수 (자원 부족)
```
