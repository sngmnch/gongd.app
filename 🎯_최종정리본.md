## 🔍 **프로젝트 핵심 내용**

### 문제의식
```
대전은 '과학도시'인데...
- 연구소, 대학, 특목고, 학원이 서구·유성구에 집중
- 같은 도시 내 교육 격차 발생
- 정보 부족 + 거리 한계 + 저참여율 = 악순환
```

### 솔루션
```
1️⃣ 지역별 교육 자원 점수화 (데이터 기반)
   - 영재고, 과학고, 특성화고, 동아리, 교원 등을 지수화
   - 구별 교육 기회 지수 계산

2️⃣ AI 기반 맞춤 추천 시스템
   - 입력: 학생의 위치 + 관심 분야
   - 출력: 추천 프로그램 + 에듀버스 정보
   - 교육 부족 지역에 가중치 부여

3️⃣ 이동 지원 (에듀버스 운영)
   - 동구·중구 중학교를 주말에 돌면서 픽업
   - 서구·유성구의 프로그램/강연장으로 이동
   - 경험 → 피드백 → 재추천 (지속적 개선)
```

## 💻 **AI 활용 계획**

### 활용 가능한 AI 도구/기법

#### 1️⃣ **데이터 분석 & 시각화**
- **생성형 AI (ChatGPT/Claude)**
  - 데이터 분석 Python 코드 생성
  - 구별 비교 분석 쿼리 작성
  - 지도 시각화 코드 (folium, plotly 등)

- **Python 라이브러리**
  - pandas: 데이터 정제 및 분석
  - matplotlib/seaborn: 그래프 생성
  - folium: 지도 시각화 (위도·경도 표시)

#### 2️⃣ **지수 계산 (AI 활용)**
```
교육 기회 지수 = 
  (영재고 수 / 지역 중학생 수) × 100
  + (과학고 수 / 지역 중학생 수) × 100
  + (평균 동아리 수) × 1
  + (과학 교원 수 / 전체 교원 수) × 50
  + (지하철 역 수) × 5

→ AI 프롬프트:
"Python으로 대전 구별 교육기회지수를 계산하는 코드를 작성해줘. 
입력은 [영재고수, 과학고수, 동아리수, 과학교원수, 중학생수]이고,
가중치는 [100, 100, 1, 50]이야."
```

#### 3️⃣ **추천 알고리즘 (AI 활용)**
- 생성형 AI로 추천 로직 작성
- 사용자 입력 (위치, 관심분야) → 프로그램 매칭
- 가중치 조정으로 교육부족지역 학생 우선

#### 4️⃣ **PPT 자동화**
- **생성형 AI로 발표자료 자동화**
  - 그래프 요약 텍스트 생성
  - 슬라이드별 설명문 작성
  - AI 기반 PPT 디자인 도구 활용
