## 🎬 **빠른 시작 (Quick Start)**

### 1단계: 환경 설정
```bash
# Node.js 18+ 설치 (https://nodejs.org)
# Git 설치

# 프로젝트 클론
git clone <repo-url>
cd science-shuttle-app

# 프론트엔드 설정
cd frontend
npm install
npm start  # localhost:3000

# 백엔드 설정 (새 터미널)
cd backend
npm install
npm start  # localhost:5000
```

### 2단계: 데이터베이스 설정
```bash
# PostgreSQL 설치 및 실행
# 데이터베이스 생성
createdb science_shuttle

# 테이블 생성 (backend/migrations 실행)
npm run migrate
```

### 3단계: 더미 데이터 입력
```bash
# 초기 학교, 프로그램 데이터 삽입
npm run seed
```

#### 2. BusBooking.jsx (셔틀 예약)
```jsx
import React, { useState, useEffect } from 'react';
import { getBuses, makeBooking } from '../services/api';

function BusBooking() {
  const [buses, setBuses] = useState([]);
  const [selectedBus, setSelectedBus] = useState(null);

  useEffect(() => {
    getBuses().then(data => setBuses(data));
  }, []);

  const handleBooking = async (busId) => {
    try {
      const result = await makeBooking(busId);
      alert('예약되었습니다!');
    } catch (error) {
      alert('예약 실패: ' + error.message);
    }
  };

  return (
    <div className="bus-booking">
      <h2>이번주 셔틀 버스</h2>
      
      {buses.map(bus => (
        <div key={bus.id} className="bus-card">
          <h3>🚌 버스 {bus.id}호</h3>
          <p>🎯 {bus.program_name}</p>
          <p>📅 {bus.date} {bus.time} 출발</p>
          <p>📍 픽업: {bus.pickup_location} ({bus.pickup_time})</p>
          
          {/* 좌석 표시 */}
          <div className="seats">
            {Array(bus.total_seats).fill(0).map((_, i) => (
              <span key={i} className={i < bus.booked_seats ? "seat filled" : "seat empty"}>
                {i < bus.booked_seats ? "🔴" : "⚪"}
              </span>
            ))}
          </div>
          
          <button 
            onClick={() => handleBooking(bus.id)}
            disabled={bus.booked_seats >= bus.total_seats}
            className="book-btn"
          >
            {bus.booked_seats >= bus.total_seats ? "만석" : "예약하기"}
          </button>
        </div>
      ))}
    </div>
  );
}

export default BusBooking;
```

## 🤖 **추천 알고리즘 구현**

### algorithms/recommender.js
```javascript
class RecommendationEngine {
  
  // 학생 프로필 기반 추천
  async getRecommendations(studentId, limit = 5) {
    const student = await this.getStudent(studentId);
    const allPrograms = await this.getAllPrograms();
    
    // 각 프로그램별 점수 계산
    const scores = allPrograms.map(program => ({
      program,
      score: this.calculateScore(student, program)
    }));
    
    // 점수 기준 정렬 후 상위 limit개 반환
    return scores
      .sort((a, b) => b.score - a.score)
      .slice(0, limit)
      .map(item => item.program);
  }

  // 추천 점수 계산
  calculateScore(student, program) {
    const interestMatch = this.calculateInterestMatch(student, program);
    const regionalPriority = this.calculateRegionalPriority(student.region);
    const difficultyFit = this.calculateDifficultyFit(student.level, program.level);
    const popularity = this.calculatePopularity(program.bookings, program.capacity);
    
    // 가중치 적용
    return (
      (interestMatch * 0.4) +
      (regionalPriority * 0.3) +
      (difficultyFit * 0.15) +
      (popularity * 0.15)
    );
  }

  // 관심분야 매칭도 (0-1)
  calculateInterestMatch(student, program) {
    const studentInterests = student.interests; // ['수학', '로봇']
    const programTopics = program.topics;        // ['로봇', 'AI']
    
    const matchCount = studentInterests.filter(
      interest => programTopics.includes(interest)
    ).length;
    
    return Math.min(1, matchCount / studentInterests.length);
  }

  // 지역 격차 우선도 (0-1)
  calculateRegionalPriority(region) {
    const regionalScores = {
      '서구': 0.3,
      '유성구': 0.4,
      '대덕구': 0.5,
      '중구': 0.9,    // 격차가 크므로 우선도 높음
      '동구': 1.0     // 가장 격차가 크므로 최우선
    };
    
    return regionalScores[region] || 0.5;
  }

  // 난이도 적합도 (0-1)
  calculateDifficultyFit(studentLevel, programLevel) {
    const levels = { '초급': 1, '중급': 2, '고급': 3 };
    const diff = Math.abs(levels[studentLevel] - levels[programLevel]);
    
    // 난이도 차이가 작을수록 점수 높음
    return Math.max(0, 1 - diff * 0.3);
  }

  // 인기도 (0-1)
  calculatePopularity(bookings, capacity) {
    return bookings / capacity;
  }
}

module.exports = RecommendationEngine;
```

## 🗄️ **데이터베이스 스키마**

### Student (학생)
```sql
CREATE TABLE students (
  id SERIAL PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  password_hash VARCHAR(255) NOT NULL,
  name VARCHAR(100) NOT NULL,
  school_id INT REFERENCES schools(id),
  grade INT,
  region VARCHAR(50),
  interests TEXT[], -- ['수학', '로봇']
  level VARCHAR(20), -- '초급', '중급', '고급'
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

### Program (프로그램)
```sql
CREATE TABLE programs (
  id SERIAL PRIMARY KEY,
  title VARCHAR(200) NOT NULL,
  description TEXT,
  topics TEXT[], -- ['로봇', 'AI']
  level VARCHAR(20), -- '초급', '중급', '고급'
  location VARCHAR(255),
  date DATE NOT NULL,
  time TIME NOT NULL,
  capacity INT,
  instructor_name VARCHAR(100),
  institution VARCHAR(100),
  bus_provided BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### Bus (셔틀)
```sql
CREATE TABLE buses (
  id SERIAL PRIMARY KEY,
  program_id INT REFERENCES programs(id),
  route TEXT NOT NULL, -- '가양중->동산중->대덕고'
  pickup_points TEXT[], -- 픽업 위치들
  pickup_times TIME[], -- 각 픽업 시간
  departure_time TIME NOT NULL,
  arrival_time TIME NOT NULL,
  capacity INT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### Booking (예약)
```sql
CREATE TABLE bookings (
  id SERIAL PRIMARY KEY,
  student_id INT REFERENCES students(id),
  bus_id INT REFERENCES buses(id),
  booking_date TIMESTAMP DEFAULT NOW(),
  status VARCHAR(20), -- 'confirmed', 'cancelled', 'completed'
  guardian_approved BOOLEAN DEFAULT false,
  cancelled_at TIMESTAMP
);
```

### Region (지역)
```sql
CREATE TABLE regions (
  id SERIAL PRIMARY KEY,
  name VARCHAR(50),
  latitude DECIMAL(10, 8),
  longitude DECIMAL(11, 8),
  gifted_schools INT,
  science_schools INT,
  avg_clubs INT,
  science_teachers INT,
  total_students INT,
  resource_score DECIMAL(5, 2)
);
```
