# 🚀 대전 수·과학 교육격차 완화 앱 - 제작 방향 & 기능 설계

---

## 📱 **앱 기본 개요**

### 앱 이름 (아이디어)
- **"사이언스 셔틀"** 또는 **"뉴톤버스"**
- 부제: "대전의 모든 학생을 위한 과학 기회"

### 타겟 사용자
- **1차**: 대전 중학교 1-3학년 학생 (만 13-15세)
- **2차**: 고등학생 (진로 탐색 학생 중심)
- **보호자**: 부모 (자녀 활동 관리)

### 앱 타입
- **웹 기반 앱** (모바일 + PC 모두 지원)
- 간단한 회원가입, 실시간 조회 기능

## 🎯 **핵심 기능 (3가지)**

### 1️⃣ **학생 맞춤형 추천 엔진**

#### 학생이 입력하는 정보 (온보딩)
```
[화면: 프로필 만들기]

Q1. 당신의 학교는? 
   → 드롭다운으로 대전 모든 중학교 리스트
   → 자동으로 "지역" 파악 (서구/동구/중구/유성구/대덕구)

Q2. 당신의 관심분야는? (중복선택 가능)
   ☐ 수학
   ☐ 물리
   ☐ 화학
   ☐ 생물
   ☐ 지구과학
   ☐ 로봇/코딩
   ☐ 환경

Q3. 주말에 이동 가능한가?
   ☐ 네, 가능합니다
   ☐ 아니오

Q4. (선택) 생기부 텍스트 붙여넣기
   → AI가 관심분야 추가 분석
```

#### AI 분석 (백엔드)
```
학생 프로필 생성:
{
  "학생_id": "STU_001",
  "학교": "대전가양중학교",
  "지역": "동구",
  "관심분야": ["수학", "물리", "로봇"],
  "이동가능": true,
  "AI분석_기타관심": ["STEM", "미래기술"],
  "우선순위": 8 (교육부족지역이므로 높음)
}

지역 격차 점수:
- 동구: 교육 자원 점수 45/100 (낮음)
→ 우선 추천 대상 ⭐
```

#### 추천 결과 (추천)
```
[화면: 당신을 위한 추천 프로그램]

🌟 우선 추천 (당신을 위해 준비했어요)
1. [7/13 오후 2시] 대덕고 "수학의 신비" 강연
   - 장소: 유성구 대덕고등학교
   - 강사: 서울대 수학과 교수
   - 탑승: 동구 가양중 (10:30 픽업)
   - 🚌 셔틀 예약 가능

2. [7/20 오전 10시] 대전과학관 "AI로봇 체험"
   - 장소: 중구 과학관
   - 참가비: 무료
   - 🚌 셔틀 예약 가능

3. [7/27] 한국전자통신연구원(ETRI) 투어
   - 장소: 유성구 대덕연구단지
   - 제약: 보호자 동반 필수
   - 🚌 셔틀 예약 불가 (학원 버스 추천)

📌 왜 이 프로그램들을 추천했나요?
   → 당신이 관심 있는 분야 + 당신 지역의 교육 격차 해소
```

## 📈 **추천 알고리즘 상세**

### 추천 점수 계산식
```
추천_점수 = 
  (관심분야_매칭도 × 0.4)
  + (지역격차_우선도 × 0.3)
  + (프로그램_난이도 × 0.15)
  + (다른학생_참여도 × 0.15)

예시:
프로그램: "로봇 만들기"
학생: 동구, 로봇 관심, 초급자

관심분야_매칭도 = 0.95 (로봇 강한 관심)
지역격차_우선도 = 0.9 (동구는 프로그램 부족)
프로그램_난이도 = 0.8 (초급 레벨)
다른학생_참여도 = 0.85 (비슷한 학생 3명 참여)

최종 점수 = (0.95×0.4) + (0.9×0.3) + (0.8×0.15) + (0.85×0.15)
         = 0.38 + 0.27 + 0.12 + 0.1275
         = 0.8775 ≈ 88점 ⭐⭐⭐⭐⭐ (강력 추천)
```

## 💡 **차별화 포인트**

### 기존 서비스 vs 우리 앱
```
❌ 기존:
   - 정보만 제공 (한 번 본 후 끝)
   - 거리 문제 해결 안 함
   - 학생 주체성 낮음
   
✅ 우리 앱:
   - 추천 → 예약 → 참여 → 피드백 (완전한 서비스)
   - 에듀버스로 거리 문제 해결
   - 학생 맞춤형 경험
```
