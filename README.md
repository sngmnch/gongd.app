# 🚌 사이언스 셔틀 - 대전 교육격차 분석 앱

제8회 교육 공공데이터 AI 활용대회 출품작

## 📊 프로젝트 개요

대전의 교육 격차를 공공데이터로 분석하고, AI 기반 추천 엔진을 통해 교육 기회를 공평하게 제공하는 모바일 앱입니다.

### 핵심 문제
- **중구**: 교원 1인당 학생 15.6명 (유성구 12.8명과 비교 시 3명 차이)
- **서구**: 교원 부족으로 인한 교육 기회 불균등
- **동구, 대덕구**: 소규모 학교로 인한 자원 부족

### 해결책
1. **AI 기반 맞춤형 추천**: 지역별 격차를 고려한 프로그램 추천
2. **실시간 셔틀 버스 예약**: 교육 자원 부족 지역 학생을 위한 셔틀 운영
3. **교육 격차 시각화**: 공공데이터 기반 통계로 문제의 심각성 입증

## 🚀 빠른 시작

### 필수 요구사항
- Node.js 18+
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# http://localhost:3000 에서 앱 확인
```

### 배포 빌드
```bash
npm run build
```

## 📊 주요 기능

### 1️⃣ 구별 고등학교 동아리 수 비교
- 각 구의 동아리 개수 시각화
- **발견**: 대덕구(64.3개) vs 유성구(45개) 격차

### 2️⃣ 교원 1인당 학생 수
- 교육 질을 나타내는 핵심 지표
- **발견**: 중구(15.6명) vs 유성구(12.8명) - 3명 차이

### 3️⃣ 지역별 교육 격차 종합 지수
- 100점 만점 종합 점수
- **우선 지원 대상**: 중구(65점) → 서구(72점) → 동구(78점)

### 4️⃣ 에듀버스 지원 필요도
- 교육 자원 점수에 따른 버스 운행 우선순위 결정
